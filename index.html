How to use?

Enter the IP and the Port of your VLC-server above. If you run the app for the first time you have to set your connection details (ip, port). Furthermore you can set a home-folder, but that is optional. Please note, that the VLC may take a moment to react to the commands. Be patient :).

The icons explained

  • Takes you back to the player

  • Repeats the file once

  • Repeats all the files

  • Shuffles through playlist

  • You can mute the volume if you tap on the percentage and unmute it if you click again
