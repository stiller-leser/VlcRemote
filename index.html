How to use?

Enter the IP and the Port of your VLC-server above. If you run the app for the first time you have to set your connection details (ip, port). Furthermore you can set a home-folder, but that is optional.
Sometime there are more options hidden behind a tap!

The icons explained

I'm not a grafic designer ;)
